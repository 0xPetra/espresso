KVMerkleTree

The core enum of the key-value Merkle tree. Making use of KVTreeHash, it is generic over branch-arity (number of branches). Note: this type implements PartialEq so that containing types can derive PartialEq, mostly for testing purposes. The implementation tests for logical equality of the represented set, ignoring sparseness. That is, any two sets with the same root hash will compare equal, even if the elements retained in memory are different between the two sets.
